# Cyber Lab

Repositorio de estudos praticos em seguranca ofensiva. Contem scripts de reconhecimento, analise de vulnerabilidades e anotacoes tecnicas organizadas por tema.

## Estrutura

```
cyber-lab/
├── recon/               # Reconhecimento e enumeracao
├── scanning/            # Varredura de portas e servicos
├── exploitation/        # Tecnicas e anotacoes de exploracao
├── post-exploitation/   # Escalacao de privilegios e persistencia
├── writeups/            # Resolucao de CTFs e labs
└── notes/               # Anotacoes tecnicas e referencias
```

## Ferramentas utilizadas

- Kali Linux
- Nmap
- Metasploit Framework
- Burp Suite
- Gobuster
- Netcat
- Python (scripts customizados)

## Plataformas

- TryHackMe
- HackTheBox
